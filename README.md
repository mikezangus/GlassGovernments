# Installation
## Setting up environment
### 1. Clone repository
```bash
git clone https://github.com/mikezangus/GlassGovernments.git
cd GlassGovernments
```
### 2. Create environment
```bash
conda env create -f environment.yml
conda activate glass_governments_env
```
